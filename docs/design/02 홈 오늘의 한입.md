**전제:** `00 공통 디자인 원칙.md`. 이 문서는 이 페이지의 목적·구성 요소만 기술한다.

# 1. 어떤 페이지인가
- **홈 화면 (오늘의 한입)** — 재방문 사용자의 기본 진입점. 오늘의 학습 세션을 시작하거나 이어서 한다.
- 도메인 근거(기능 1.5 일일 세션 · 기능 6 지속·성취 동기): 세션은 **문제 분량 기반**(시간 아님), 서비스가 **새+복습을 자동 배정**, 중단해도 **재개** 가능. 재접속 동기(스트릭 등 긍정 신호)도 여기서 보여준다.
- 목적: "오늘의 한입"이라는 **작고 명확한 목표 하나**를 제시해 부담 없이 시작하게 한다(1 Page = 1 Action).

# 2. 이 페이지의 테마 강조점
- **가벼운 초대.** "오늘도 한입 할까요?" 톤. 숙제·의무감 아닌 습관 유도(감정적 압박 금지).
- 진행은 **분량**으로(예: 오늘 5문제 중 2개 완료). **카운트다운 타이머 금지.**
- 완료했으면 **긍정 빈 상태**("오늘 몫은 다 했어요! 더 할 수도 있어요").

# 3. 구성 요소 (구체)
1. **인사·오늘의 한입 헤더** — "오늘도 한입 해볼까요?" + 날짜감(선택).
2. **세션 진행 요약** — 오늘 세션의 본 문제 진행(예: `2 / 5`, 진행 막대). 분량 기반, 시간 아님.
3. **Primary CTA — 상태에 따라 문구 전환**
   - 미시작: **[오늘의 한입 시작하기]**
   - 진행 중(중단 후 재방문): **[이어서 풀기]** — 남은 본 문제·진행 지점으로 복귀.
   - 오늘 완료: **[조금 더 풀어보기]**(GhostButton, 압박 아님). **[반영 2026-07-19: D6]** 눌러도 새 세션이 시작될 뿐이며(추가 학습 폐지·세션으로 일원화), '오늘 완료' 배지는 오늘 완료한 세션이 1개 이상이면 그대로 유지된다. ⭐️ 완료 축하는 별도 카드가 아니라 세션 진행 요약 카드(2) 자체의 시각 변화로 표현한다(2026-07-16 오너 결정, 홈 복잡도 감소) — 배지가 완료 표식(예: "✓ 완료")으로, 진행 막대가 success 색으로 바뀌고 긍정 빈 상태 문구가 카드 안에 붙는다.
3-b. **면접 연습(면접 세션) 진입 카드 [반영 2026-07-22: 면접 대비 가치 계획 §4.3]** — 오늘의 한입 카드(세션 진행 요약(2)+Primary CTA(3))보다 **아래 위계**로 둔다. 핵심 가치의 계단 위계(1단=진입·용어 정착이 히어로, 2단=면접 세션)를 화면에서도 지킨다는 뜻이다. secondary 카드(BcsCard) — 이 카드의 CTA는 Primary가 아니다(홈의 유일한 Primary는 오늘의 한입 CTA). 서버의 `GET /api/interview/status` 응답 하나로 아래 4개 상태를 그대로 반영한다(클라이언트가 재판단하지 않음):
   - **게스트** — 가입 유도. "가입하면 지금까지 익힌 개념 N개를 면접처럼 연습할 수 있어요"(N=게스트의 현재 후보 개념 수). 누르면 05 로그인·가입으로. 강요 톤 없음(무낙인·권유).
   - **후보 0개(아직 승급된 개념 없음)** — 긍정 빈 상태. "오늘의 한입에서 개념을 스스로 맞히면 면접 연습이 열려요." 비난 없이 오늘의 한입을 계속하게 자연스럽게 유도.
   - **잔여 있음(오늘 쿼터 남음)** — 진입 CTA. "익힌 개념 N개, 면접처럼 설명해보기" → 08 면접 세션 화면.
   - **오늘 소진(쿼터 다 씀)** — 담백 안내. "내일 다시 열려요." 재촉·아쉬움 강조 없음.
4. **복습 편입 안내(은은)** — 오늘 세션에 복습이 섞였음을 부담 없이 암시("오늘은 복습도 한 스푼 섞였어요"). 사용자가 의식 안 해도 되게 강조 X.
5. **연속 학습(스트릭) 표시 — 긍정 동기(기능 6, 리뷰 반영)** — 재접속 동기의 핵심이므로 **독립 카드로 승격**해 성취로 또렷이 보여준다(2026-07-16 오너 결정, MVP 포함). 카드는 불꽃 아이콘 타일 + "N일째 꾸준히 한입!" 한 줄. ⚠️ 끊겼다고 죄책감·상실 공포 주는 연출 금지(UX 4 다크 패턴) — 끊겨도(0일) 불꽃·주황 톤 없이 "오늘 한입으로 연속 학습을 시작해요" 톤. ⚠️ 시안 초안의 "내일도 오시면 N일 연속이에요" 같은 미래 문구는 **쓰지 않는다** — 지키지 못하면(내일 못 오면) 실망을 예약하는 손실 프레임이라는 우려로 뺐다(2026-07-16 오너 결정). 오늘 성취를 말하는 한 줄만 남긴다.
6. **하단 내비게이션(있다면)** — 홈 / 계정. 최소 구성.
7. **계정 진입점** — 게스트면 "가입하고 기록 지키기" 은은한 배너(06/05로). 가입자면 프로필/설정(06).
8. **스크랩 목록 진입점(리뷰 반영, 2026-07-16 오너 결정)** — 내가 저장한 문제를 다시 보는 목록으로. 세션 진행 요약 카드 아래 조용한 secondary 행("스크랩한 문제" + `›`)으로 둔다. 눈에 띄되 Primary(오늘의 한입 CTA)를 방해하지 않게, 회원 여부와 무관하게 항상 노출한다.

# 인터랙션 흐름
1. 앱 재실행 → 홈.
2. **[시작/이어서 풀기]** → 03 문제 풀이.
3. 세션 완료 시 04 세션 완료로 이어짐(또는 홈에서 완료 상태 표시).
