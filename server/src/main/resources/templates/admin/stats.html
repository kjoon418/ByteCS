테스터 참여 지표

각 지표는 중복을 제거한 사용자 수(DISTINCT)입니다.

참고: 풀이 시작 기록은 클라이언트가 진입 시 서버에 별도로 알리는 방식(fire-and-forget)이라, 네트워크 유실 가능성으로 실제보다 적게 집계될 수 있는 하한값입니다.

전체 기간으로

아래 참여 지표(1~4)는 시작 ~ 종료 (KST) 기간의 집계입니다. 규모·난이도 지표는 시점 개념이 없어 전체 기간 기준입니다.

아래 참여 지표는 전체 기간 집계입니다. 위에서 시작·종료일을 지정하면 그 기간만 집계합니다.
